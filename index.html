Future Dream Demo

Spring Boot 기반 학습용 게시판 예제 프로젝트입니다. 이 저장소는 로컬에서 실행하거나 GitHub Pages로 간단한 프로젝트 소개 페이지를 제공할 수 있습니다.

로컬에서 실행 (개발)

프로젝트가 demo 폴더에 있습니다. 빌드 및 실행 방법:

cd demo
mvn -DskipTests package
/usr/lib/jvm/java-17-openjdk-amd64/bin/java -jar target/demo-0.0.1-SNAPSHOT.jar

브라우저에서 http://localhost:8080로 접속하세요.

GitHub Pages 배포(간단)

이 파일을 루트에 두면 GitHub Pages에서 정적 소개 페이지로 사용 가능합니다.

  • 프로젝트 페이지: `gh-pages` 브랜치로 푸시하거나
  • 또는 `Settings → Pages`에서 `main` 브랜치의 루트(root)로 설정

간단한 명령 (gh-pages 브랜치 생성 예):

git checkout -b gh-pages
git add index.html
git commit -m "Add GitHub Pages index"
git push -u origin gh-pages

프로젝트 링크

코드 위치: /demo 폴더 — 주요 파일:

  • `src/main/java/com/example/demo` — Spring 애플리케이션 소스
  • `src/main/resources/templates` — Thymeleaf 템플릿
  • `pom.xml` — Maven 설정

빠른 액션:

GitHub 저장소 보기 로컬 데모 폴더 열기
이 페이지는 저장소 루트에 위치한 정적 인덱스입니다. 동적 애플리케이션은 GitHub Pages에서 직접 실행되지 않습니다 — 프로젝트는 서버(예: Heroku, Render, VPS)에 배포해야 합니다.
